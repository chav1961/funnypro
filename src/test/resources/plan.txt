Работы по Funny Prolog 0.0.2

1. Рефакторинг проекта под применение новых возможностей Pure Library
2. Разработка SPI-интерфейса для использования его в программах на Java
В проекте funnyprolog необходимо провести ревизию кода, чтобы добавить в него все новые доработки. Для синтаксического дерева необходимо 
построить "аппаратно" реализуемые узлы, выполняющие обработку стандартных предикатов. GUI-интерфейс необходимо переработать под новые 
возможности lib (в частности, дополнить справочной системой)

Репозитарий фактов-правил содержит для каждой пары арность/ид соответствующее B+-дерево. Ключами в нем являются номера записей, причем они идут в обе стороны от нуля.
