Funny Prolog is the simple Prolog interpreter, which syntax is based on I.Bratko book. It includes into Java virtual machine via the SPI mechanism and can be used directly by javax.script package abilities. It contains a virtual Funny Prolog machine, named FProVM. Main concepts of the FProVM described in the Wiki of this projects.

This interpreter not contains and will not contains any object paradigm, exception processing etc, because this interpreter in an addition, but not replacement of the beautiful Java language. Instead, it is accurately incorporated into the Java VM environment to support easy-to-use access to the Prolog language abilities

This project uses Pure library project to support some internal functionality
